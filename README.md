# GPA Calculator Web App

A dynamic and user-friendly web application for calculating **GPA, SGPA, and CGPA**. This app allows students to manage their academic performance efficiently by providing multiple calculators and detailed academic insights.

---

## Features

- **Dynamic Subject & Semester Input:**  
  Add as many subjects and semesters as needed. The app automatically calculates your GPA based on your inputs.

- **Multiple Calculators:**  
  - **SGPA Calculator:** Calculate your **Semester Grade Point Average** for individual semesters.  
  - **CGPA Calculator:** Calculate your **Cumulative Grade Point Average** across all semesters.

- **Basic GPA Information:**  
  Provides an explanation of GPA, its purpose, and how it is calculated.

- **Interactive and Easy-to-Use Interface:**  
  Simple layout with clear input fields and buttons to dynamically add or remove subjects.

- **Web-Friendly Design:**  
  Works on both desktop and mobile devices.

---

## Usage

1. Open the `index.html` file in a browser.
2. Navigate between **SGPA** and **CGPA** calculators using the links provided.
3. Add subjects for each semester dynamically.
4. Enter grades and credit hours for each subject.
5. Click **Calculate** to get your GPA, SGPA, or CGPA.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (for dynamic subject and semester handling)

---

## How It Works

- The **SGPA** calculator computes the grade point average for a single semester.  
- The **CGPA** calculator aggregates the SGPA values from multiple semesters to give a cumulative result.  
- Users can dynamically add or remove subjects, making it flexible for any academic program.
